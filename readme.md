## To run

- `yarn`
- `yarn start`

## Methods

- `POST "/"`<br>&nbsp;&nbsp;Upload new files<br>&nbsp;&nbsp; Multipart-form data

```
files: <files>
```

- `GET "/:fileName"`<br>&nbsp;&nbsp;Retrieve file
